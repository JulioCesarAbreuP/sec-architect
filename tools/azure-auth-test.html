Cómo configurar la App Registration

  1. Crear App Registration en Entra ID > App registrations con name: sec-architect-local
  2. Redirect URI → plataforma Single-page application (SPA) → añadir la URI que aparece arriba en el campo "Redirect URI".
  3. API permissions → Add permission → Microsoft Graph → Delegated → Policy.Read.All, openid, profile
  4. Authentication → habilitar Access tokens y ID tokens (implicit grant no necesario para PKCE, pero ID token sirve para el UPN).
  5. Copiar Application (client) ID y Directory (tenant) ID desde Overview y pegarlos en el formulario.
  6. Pulsar Conectar con Microsoft — el flujo PKCE no requiere client secret.
